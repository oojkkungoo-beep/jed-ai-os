### คลิปที่เกี่ยวข้อง
| # | ชื่อคลิป | URL |
|---|---------|-----|

### Core Concepts
- **Deep Work** (Cal Newport) = งานที่ต้องสมาธิสูง สร้าง value สูงสุด
- Shallow Work = งาน reactive (email, meeting) ที่ทดแทนได้ง่าย
- AI ช่วยกำจัด shallow work → เพิ่มเวลา deep work

### ใช้ทำอะไรได้
- [ ] ใช้ Laura OS กำจัด shallow work ทั้งหมด → โฟกัส deep work
- [ ] Design วันทำงานให้มี deep work block ที่ชัดเจน
- [ ] วัด: งานที่ทำวันนี้ มีกี่ % เป็น deep work
