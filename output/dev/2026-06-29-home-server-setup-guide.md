# 🔧 Home Server เล็กๆ ราคาประหยัด ทนๆ — Step-by-Step Guide

## วิเคราะห์
Jed ไม่เคยใช้ terminal มาก่อน เป้าหมายคือให้ Jed ทำเองแค่ 3 จุดที่ต้องลงมือทางกาย (เสียบ USB, ติดตั้ง OS เบื้องต้น+ติ๊ก SSH, รันคำสั่ง Tailscale 1 บรรทัด) ส่วนที่เหลือทั้งหมด (CasaOS, Pi-hole, Home Assistant, Syncthing) ทีม AI เข้าไปทำให้ผ่าน SSH ได้

---

## ส่วนที่ 1 — งานของ Jed (ทำครั้งเดียว ทำตามรูป/ขั้นตอนได้)

### Step 1: ซื้อฮาร์ดแวร์ (งบ ≤3,000 บาท)
- มินิพีซีมือสอง: **Lenovo ThinkCentre M73/M93 Tiny** หรือ **Dell OptiPlex 3020/7020 Micro** (i5 gen4 ขึ้นไป, RAM 8GB) ~2,000-2,500 บาท
- SSD ใหม่ 120-256GB ~400-600 บาท (ถ้าเครื่องเดิมยังเป็น HDD หรือ SSD เล็กเกินไป)
- ซื้อจาก: กลุ่ม Facebook ขายคอมมือสอง, Shopee/Lazada ร้านมีรีวิวคอมมือสอง, หรือร้านคอมมือสองแถวบ้าน
- **เช็คก่อนซื้อ:** เปิดเครื่องติดไหม, พอร์ต USB/LAN ใช้ได้ครบ, ไม่มีจอฟ้า/เสียงพัดลมดังผิดปกติ

### Step 2: เตรียม USB ติดตั้ง Ubuntu Server
1. ดาวน์โหลด **Ubuntu Server LTS** (ISO) จาก ubuntu.com/download/server — ใช้คอม/โน้ตบุ๊กที่ใช้อยู่ปัจจุบัน
2. ดาวน์โหลดโปรแกรม **Rufus** (rufus.ie) — โปรแกรม GUI ทำ USB บูตได้ ไม่ต้องพิมพ์คำสั่ง
3. เสียบ USB (อย่างน้อย 8GB) → เปิด Rufus → เลือกไฟล์ ISO ที่โหลดมา → กด Start
4. รอจนเสร็จ (Rufus จะมี progress bar ให้ดู)

### Step 3: ติดตั้ง Ubuntu Server บนมินิพีซี
1. เสียบ USB เข้ามินิพีซี เปิดเครื่อง กดปุ่มเข้า Boot Menu (ปกติคือ F12 หรือ F2 — ขึ้นโลโก้ตอนเปิดเครื่องจะมีบอก) เลือกบูตจาก USB
2. หน้าจอติดตั้งจะถามทีละขั้น เลือก/พิมพ์ตามนี้:
   - ภาษา/Keyboard: English เลย
   - Network: ถ้าเครื่องต่อสาย LAN จะขึ้น IP ให้อัตโนมัติ (**จดเลข IP นี้ไว้ เช่น 192.168.1.50** — ต้องใช้ทีหลัง)
   - ตั้ง **Profile**: ใส่ชื่อผู้ใช้ + รหัสผ่าน (**จำให้แม่น จะใช้ login**)
   - **สำคัญที่สุด:** หน้า "SSH Setup" → ติ๊ก ✅ **"Install OpenSSH server"** (ถ้าไม่ติ๊กตรงนี้ ทีม AI จะเข้าไม่ได้)
   - หน้า "Featured Server Snaps": ข้ามได้ ไม่ต้องเลือกอะไร
3. รอติดตั้งเสร็จ → เครื่อง reboot → ดึง USB ออก → เครื่องจะบูตเข้า Ubuntu Server (จอจะเป็นตัวอักษรล้วน ไม่มี GUI — ปกติ ไม่ต้องตกใจ)

### Step 4: บอกข้อมูลให้ทีม
ส่งข้อความบอก Laura/Cinder แค่ 3 อย่าง:
- IP ของเครื่อง (จาก Step 3)
- Username ที่ตั้งไว้
- Password ที่ตั้งไว้

→ จากนี้ **ทีม AI เข้า SSH ทำงานต่อให้ทั้งหมด** (ถ้าเครื่อง Jed ใช้รัน Claude Code อยู่ในเครือข่ายเดียวกับมินิพีซี จะ SSH เข้าตรงได้เลย)

## ส่วนที่ 2 — งานที่ทีม AI ทำให้ (ผ่าน SSH, Jed ไม่ต้องพิมพ์อะไร)

Cinder จะ SSH เข้าไปรันให้ครบตามลำดับนี้:

1. **อัปเดตระบบ**: `apt update && apt upgrade -y`
2. **ลง Docker**
3. **ลง CasaOS** (one-line installer) — ได้ dashboard หน้าเว็บ เข้าผ่าน `http://<IP-เครื่อง>` จากเบราว์เซอร์ในบ้าน
4. **ติดตั้ง app ผ่าน CasaOS App Store**:
   - Syncthing (file sync/backup)
   - Pi-hole (บล็อกโฆษณาทั้งบ้าน — ต้องตั้ง router ให้ใช้ Pi-hole เป็น DNS ด้วย ทีมจะบอกขั้นตอนเพิ่ม)
   - Home Assistant (smart home)
5. **ติดตั้ง Tailscale** — ขั้นตอนนี้ต้องให้ Jed กด authorize ผ่านลิงก์ที่ทีมส่งให้ (เปิดในเบราว์เซอร์ คลิกเดียว ไม่ต้องพิมพ์คำสั่ง)
6. **ตั้งค่าพื้นฐาน** + ทดสอบแต่ละ service

## ส่วนที่ 3 — สิ่งที่ Jed ต้องทำเพิ่มอีกนิดหลังระบบพร้อม
- ลงแอป **Tailscale** บนมือถือ/โน้ตบุ๊กที่อยากใช้เข้าถึงจากนอกบ้าน → login ด้วยบัญชีเดียวกัน (1 คลิก authorize)
- ลงแอป **Syncthing** บนมือถือถ้าต้องการ sync ไฟล์อัตโนมัติ
