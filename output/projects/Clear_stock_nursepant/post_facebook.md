# โพสต์ Facebook — เคลียร์สต๊อก NursePant (เวอร์ชันล่าสุด)

🔔 **เคลียร์สต๊อกครั้งเดียว — กางเกงพยาบาล NursePant**

ของใหม่ 100% ไม่เคยผ่านการใช้งาน ผลิตมาพร้อมกันทั้งล็อต เก็บสต๊อกไว้พักหนึ่ง ขอเคลียร์ในราคาพิเศษให้พยาบาล/นักศึกษาพยาบาลที่กำลังมองหากางเกงทำงานดีๆ ในราคาคุ้มสุดๆ ครับ

💰 **ราคาเดียวทุกไซส์ 420 บาท** (จากราคาปกติ 750 บาท)
🚚 ราคานี้ยังไม่รวมค่าส่ง

📋 **สต๊อกคงเหลือ (อัปเดต ณ วันนี้):**
- Size S — 2 ตัว
- Size M — 9 ตัว
- Size L — 18 ตัว
- Size XL — 16 ตัว
- Size XXL — 17 ตัว
- Size XXXL — 4 ตัว

⚠️ ของมีจำนวนจำกัดตามจำนวนจริงด้านบน หมดแล้วหมดเลย ไม่มีรอบผลิตเพิ่ม

สนใจไซส์ไหน ทักแชทมาได้เลยครับ 🙏
