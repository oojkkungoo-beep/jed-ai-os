# วิธีใช้โฟลเดอร์ team/

1. คัดลอก `_persona_template.md` เป็นไฟล์ใหม่ต่อ agent 1 คน เช่น `idea.md`, `life.md`, `finance.md`
2. แก้เนื้อหาในไฟล์ให้ตรงกับหน้าที่ของ agent นั้น
3. กลับไปแก้ตาราง Routing ใน `CLAUDE.md` (root) ให้ชี้มาที่ไฟล์นี้
4. ไม่จำเป็นต้องมีครบทุก agent ตั้งแต่แรก — เริ่มจาก 2-3 ตัวที่ใช้บ่อยที่สุดก่อน แล้วค่อยเพิ่ม

## แนวคิดการแบ่ง agent
- แบ่งตาม "หมวดชีวิต/งาน" ที่คุณต้องจัดการบ่อยๆ (เช่น งาน, เงิน, สุขภาพ, ไอเดีย)
- แต่ละ agent ควรมี scope ชัดเจน ไม่ทับซ้อนกันมากเกินไป
- ถ้างานไหนไม่เข้าหมวดไหนเลย ให้ orchestrator (CLAUDE.md หลัก) จัดการเองหรือเป็น "งานเล็ก/ด่วน"
