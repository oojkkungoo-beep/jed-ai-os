# 📔 Diary — วันพฤหัสบดีที่ 18 มิถุนายน 2569

## งานที่ทำวันนี้

**ชศพอ. Web App — ยกเครื่องหน้าค้นหาสมาชิก + Admin Panel:**
- หน้าค้นหาสมาชิก: ค้นหาสดขณะพิมพ์, dropdown รุ่น/ประเภทดึงค่าจริงจากข้อมูลแบบไม่ซ้ำ + ค้นหาอัตโนมัติเมื่อเลือก, compact row ใน scroll box แสดงผลทุกรายการ
- Admin panel: หน้า "ภาพรวม" เป็น dashboard สถิติ + ย้ายการอนุมัติใบสมัครมาไว้ในหน้าเดียว, หน้า "จัดการสมาชิก" คลิกแถวดูรายละเอียด + แก้ไข/ลบ + ดู/เพิ่ม Logbook รายคน, Logbook สวัสดิการคลิกดูรายละเอียด แก้ไข/ลบได้
- ปรับ form ใบสมัคร: คำนำหน้า/รุ่นเป็น text อิสระ, dropdown สถาบัน + ประเภทสมาชิก, ตัด section ที่อยู่บ้านออก, ออกแบบให้ข้อมูลสมัครใหม่ต่อเข้าตาราง Members ได้
- Export Excel เปลี่ยนเป็นโหลดลงเครื่องตรง ไม่ทิ้งไฟล์ค้างใน Drive
