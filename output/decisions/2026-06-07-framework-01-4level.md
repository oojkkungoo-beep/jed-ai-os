## 📐 สรุป Framework (4 Level) — ตกลงกันแล้ว

**ลำดับความสำคัญตามที่ Jed เลือก (มาก→น้อย):** ครอบครัว > เงิน > ความคิดวนซ้ำ > สุขภาพ > งาน

**โฟกัส 90 วันข้างหน้า:** เงิน > ครอบครัว > สุขภาพ

1. **ตัวเอง** — Wealth + Health + Mindset (จุดเริ่มต้น, ฐานราก)
2. **ครอบครัว** — ความสุข ความมั่นคง ความอบอุ่น
3. **งาน/อาชีพ** — เติบโต มีผลงานภาคภูมิใจ
4. **ธุรกิจ** — สร้างองค์กร สร้าง legacy

## สถานะล่าสุด (อัปเดตจากความคืบหน้าหลังจากนี้)
- ✅ Level 2 (ครอบครัว): ปลดล็อกแล้ว (2026-06-08) — ภรรยาช่วยพาลูกเข้านอน + สลับเวรกันเอง
- 🔄 Level 1 (ตัวเอง — เงิน): กำลังดำเนินแผนรายได้เสริม 90 วัน (NursePant clear stock เป็น quick win แรก) — ดูรายละเอียดใน memory `project_pending_tasks`
- ⏳ Level 1 (ตัวเอง — สุขภาพ): ใช้ "Minimum Viable Day" framework แทนสัดส่วนเวลาตายตัว, Eir (agent ใหม่) ดูแลด้าน health
- ⏳ Level 3-4 (งาน/ธุรกิจ): ยังไม่เริ่ม

**Why:** เก็บเป็นเอกสารอ้างอิงหลักของ framework ชีวิต Jed ในโปรเจกต์ (ก่อนหน้านี้มีแต่ใน memory agent) เพื่อให้ทุก agent ในทีม (Atlas, Sage, Eir, Mint ฯลฯ) อ้างอิงจุดเดียวกันได้
**How to apply:** เมื่อมีความคืบหน้าระดับใด ให้กลับมาอัปเดต "สถานะล่าสุด" ในไฟล์นี้ — ไม่ต้องสร้างไฟล์ framework ใหม่ซ้ำ
